# SnapLocal MVP Test Objectives
Version: 1.0 (April 2026)
Goal: Every demo website must feel like a real, conversion-optimized local business site generated in <30 seconds.

## Core Requirements (apply to ALL 5 demos)
- Next.js 15 App Router + TypeScript + Tailwind + shadcn/ui
- Fully mobile-first, loads <1s on mobile
- Modern, clean, professional design (no generic templates)
- Hero section with high-quality placeholder image + business name
- Prominent primary CTA: “Book Now” / “Reserve Table” (opens booking modal)
- Floating WhatsApp chat button (bottom-right, opens wa.me link with pre-filled message)
- Sticky mobile bottom bar with “Book Now” + WhatsApp
- Google Maps embed + full address
- “Leave a Google Review” button (links to fake Google review URL)
- Fake analytics teaser in footer (“See your live traffic dashboard”)
- SEO-ready: proper <title>, meta description, Open Graph tags
- All interactive elements (booking, WhatsApp) must actually work
